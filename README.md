# Ministry-of-Infrastructure-project-transport-networks
"This report presents a sample work for Department of Transport Strategy as part of the 'STARTUJ Z naMI' program. It includes an Excel table for monitoring two infrastructure projects, delay calculations using formulas, conditional formatting, and statistical analysis. Recommendations for sustainable transport development in Poland are provided."

Analyzed two projects: "Hydrogen Refueling Station Construction" and "Works on C-E20 Pilawa-Łuków section".
Monitored milestones: tender announcement, building permit, usage permit.
Used MS Excel for tracking delays with formulas (IF, DATE.DIFF) and conditional formatting.
Created statistical analysis table with interpretations (mean, median, quartiles, etc.).
Included recommendations for sustainable transport development in Poland.
